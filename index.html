Medeco Flutter App UI Kit

Medec Health App UI Kit complete UI project using Flutter, it contains more than 20 screens ready to integrate with your backend (Firebase, PHP, Java, ASP...) projects, you can reuse more than 60 widgets to customize your application.The same code used for both iOS and Android

Getting Started with UI Kit

  • After download and unzip the package, use preferred IDE (Android Studio / Visual Code / IntelliJ) to open the project medec_health_flutter_ui_by_smartervision folder.
  • Click the Get dependencies or Packages get to install the libraries from pubspecs.yaml file.
  • Open the simulator to run iOS or Android (as the step above)
  • Then press the run button to start project (you can still open multi simulator at the same time)
